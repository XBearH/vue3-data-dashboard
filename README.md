# Vue3 数据看板 Dashboard 
一个基于 Vue3 + Element Plus + TypeScript 开发的响应式数据看板，适用于后台管理系统首页展示。

## 技术栈
- **Vue 3** (Composition API)
- **TypeScript**
- **Element Plus**
- **Vite**

## 功能亮点
- 📊 数据统计卡片（支持增长趋势）
- 📋 热门应用 / 模板 TOP5
- 🧩 快捷功能入口
- 📱 全响应式布局（适配 PC / 移动端）
- ✨ 精美 hover 交互动画

## 本地运行
```bash
npm install
npm run dev
